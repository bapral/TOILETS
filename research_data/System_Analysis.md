# 全國公廁地圖 - 系統分析報告 (最新版)

## 1. 系統架構
本系統採 **Serverless PWA 架構**，將 2.5 萬筆經處理後的公廁資料封裝於靜態 JSON 中，透過 Service Worker 實現 100% 離線可用。

## 2. 定位模式邏輯 (Core Logic)
系統區分為「自動」與「手動」兩種互斥模式，確保座標基準點唯一：
- **自動模式 (Auto GPS)**：以裝置 GPS 為準，紅點標示，畫面隨位移自動捲動。
- **手動模式 (Manual)**：
    - **等待態**：點擊「手動定位」按鈕觸發，紅點消失，等待使用者點擊地圖。
    - **鎖定態**：選點後顯示藍色 📍 並鎖定坐標，防止誤觸位移。

## 3. 搜尋與導航演算法
- **距離計算**：採用 Haversine Formula 計算經緯度直線距離。
- **最近搜尋**：遍歷 `toilets.json` 數組，比對當前基準點 (Active Marker) 的最小距離。
- **動態渲染**：根據 `zoomend` 與 `moveend` 事件，依縮放層級 (14/16/18) 動態調整標記顯示上限 (300/600/1000)，解決大範圍顯示效能問題。

## 4. UI/UX 設計規範
- **配色**：深黃 (#fbc531) 與純黑強對比，適配戶外強光。
- **圖標**：深藍路牌風格 (#0056b3) + 白色 WC 圖示，高辨識度。
- **連線**：雙層實線發光效果 (Glow Path)，導航綠 (#2ecc71) 與選取橙 (#f39c12)。
