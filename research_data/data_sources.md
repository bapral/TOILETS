# 全國公廁建檔資料 (FAC_P_07) 欄位規格書

## 1. 資料集來源
- **提供單位**: 環境部環境管理署
- **更新頻率**: 每日同步
- **資料特性**: 涵蓋全台公私立對外開放之列管廁所。

## 2. 核心欄位定義 (原始)
| 欄位 | 名稱 | 處理方式 |
| :--- | :--- | :--- |
| `number` | 建檔編號 | 作為聚合唯一的 Key |
| `name` | 公廁名稱 | 顯示於 UI 標題 |
| `latitude` | 緯度 | 轉 Float，用於地圖定位 |
| `longitude` | 經度 | 轉 Float，用於地圖定位 |
| `type` | 設施類型 | 轉換為設施標籤 (男/女/無障礙) |
| `diaper` | 尿布檯 | "1" 代表 True, "0" 代表 False |

## 3. 聚合邏輯 (Aggregation)
為解決同地點重複標記問題，系統以 `緯度(5位)+經度(5位)+名稱前5字` 進行雜湊聚合，將單元廁所合併為「站點」。
