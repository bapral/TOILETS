## 一、專案簡介
*   **目標**：提供極速、高對比、且支援全台 2.5 萬筆公廁資料的查詢服務。
*   **核心技術**：HTML5, CSS3 (高對比設計), Leaflet.js (地圖), Service Worker (離線支援), Python (資料清洗)。
*   **架構**：Serverless 靜態架構，資料以 JSON 封裝於前端，確保在無網路環境下仍可運作。

## 二、安裝指南 (PWA Setup)
本系統無須透過 App Store 或 Google Play 下載，請依以下步驟安裝至手機主畫面：

### 📱 iOS (iPhone/iPad)
1. 使用 **Safari** 瀏覽器開啟應用程式網址。
2. 點擊瀏覽器底部的 **「分享」** 圖示（向上箭頭）。
3. 在選單中向下滑動，選擇 **「加入主畫面」**。
4. 點擊「新增」，圖示即會出現在您的桌面。

### 🤖 Android
1. 使用 **Chrome** 瀏覽器開啟網址。
2. 點擊右上角「三個點」選單或觀察底部跳出的 **「將公廁快找新增至主畫面」** 提示。
3. 點擊 **「安裝」**。

## 三、使用者操作指南

### 1. 定位模式
系統提供兩種定位基準，確保在移動中或預先搜尋時皆能精準作業：
*   **自動 GPS 模式 (🔴 紅色大頭針)**：
    *   **啟動**：點擊右下角 [🎯] 按鈕。
    *   **邏輯**：地圖中心會隨您的位置移動，紅點即為您當前的 GPS 座標。
*   **手動選點模式 (🔵 藍色標記 📍)**：
    *   **啟動**：點擊頂部標題列的 **「手動定位」**。
    *   **操作**：直接點擊地圖上任一位置。系統會鎖定該點為基準點，不再隨 GPS 移動，適合查詢遠方公廁。

### 2. 導航與連線
*   **最近目標 (綠色連線)**：
    *   點擊右下角 [🚀] 按鈕，系統會自動連接距離基準點最近的公廁。
*   **自選目標 (橙色連線)**：
    *   點擊地圖上的 **深藍路牌圖標**。系統會顯示該站點資訊並畫出導航線。

### 3. 詳細資訊面板
點擊公廁圖標後，底部會彈出面板，顯示：
*   **地址與名稱**：長按文字可複製（方便轉傳）。
*   **設施清單**：顯示男廁、女廁、無障礙、尿布檯、性別友善等。
*   **距離與時間**：顯示步行至該處的估計時間。

## 四、資料維護與更新 (ETL)
管理員可透過 Python 腳本定期更新環境部 API 的原始資料。

### 1. 環境需求
*   安裝 Python 3.8+。
*   安裝必要套件：`pip install requests`。

### 2. 一鍵更新流程 (推薦)
在專案根目錄執行以下命令，系統會自動完成「抓取 -> 清理 -> 封裝 -> 更新版本」：
```bash
python build_app.py
```

### 3. 分步指令 (進階)
若需個別處理資料，可使用以下腳本：
*   **抓取資料**：`python fetch_all_toilets.py` (從環境部抓取 `all_toilets.json`)。
*   **資料清洗**：`python clean_toilets.py` (執行座標聚合與無效資料剔除)。

## 五、技術規格與系統分析

### 1. 座標聚合演算法
為避免地圖上標記重疊（例如同一棟建築有男廁、女廁、無障礙三筆資料），系統會依據 **(緯度.5f, 經度.5f, 名稱前5字)** 進行 Hash 聚合，將設施合併至單一站點。

### 2. 離線機制 (Offline Logic)
*   **Service Worker**：所有靜態檔案 (`index.html`, `toilets.json`, 圖示) 均會被快取。
*   **版本控制**：每次執行 `build_app.py` 時，`sw.js` 內的 `CACHE_NAME` 會隨時間戳更新，強制手機端在下次連網時下載最新公廁數據。

### 3. 視覺設計規範
*   **對比度**：背景使用 `#fbc531` (鮮黃)，文字為黑色，確保在強光戶外環境下仍具備極高易讀性。
*   **導航線**：實線具備雙層光暈（2px 核心 + 6px 光暈），提升視覺深度。
